# 🌍 Pax Historia — Военно-Политическая Ролевая Игра

Добро пожаловать на официальный сайт проекта **Pax Historia**! Это текстовая военно-политическая игра (ВПИ), где каждый игрок может возглавить государство, управлять армией, вести дипломатию и переписать историю.

---

## 📜 Основные правила игры

1. **Уважение и фэйр-плей:** Уважайте других игроков и администрацию. Любые конфликты решаются через главных судей.
2. **Реализм действий:** Все ваши приказы, реформы и военные действия должны быть логически обоснованы и соответствовать исторической эпохе.
3. **Ходы и приказы:** Игровые ходы сдаются в специальную форму или тему в группе согласно установленному дедлайну.
